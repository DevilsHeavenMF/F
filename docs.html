Adding to group

Use below steps to add Bolt Backer your group.

  1. Go to Bolt Backer and send /start
  2. Then you can see a button called Add to group.
  3. Click on that button and select your groups.

To use Bolt Backer in your group, Bolt must be an admin of your group. Ti do that use below steps.

  1. Go to your group and click the three dots.
  2. Select Manage group option.
  3. Then go to Administrators.
  4. Finally click on Add Administrator and select Bolt Backer.

Using Commands

Here are the main commands that can be used with Bolt Backer

  • /start - Start to Bolt Backer
  • /help - To get full help menu.

For a easy and fully help you can use /help command in Bolt Backer bot's chat.
